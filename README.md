## How to use



clone this repo:
```sh
git clone https://github.com/electrokido321/web-checker.git
```
go in to the directory:
```sh
cd web-checker
```
enter your websites:
```sh
nano websites.txt
```
- Then enter all your website's
And exit with CTRL+X and press enter

install requests:
```sh
pip install requests
```
start the script:
```sh
python3 main.py
```
